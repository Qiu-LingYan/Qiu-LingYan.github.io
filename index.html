如何使用Hexo搭建个人博客

Posted on 2022-11-26

前言

&emsp;关于我使用Hexo搭建了个人博客以后，萌生了写作的念头。虽然小张经常骂我，但是我很生气我并没有恨他，因为他对我的编程之路有很大的帮助。今天，我就来跟风撰写一篇如何搭建个人Blog的文章。

前提

&emsp;你需要安装好Nodejs以及Git.

正文

安装cnpm（淘宝源）

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo-cli

cnpm install -g hexo-cli

&emsp;可以使用 hexo -v 来验证下。

新建文件夹

&emsp;在指定位置新建文件夹，名字为blog。

初始化博客

hexo init

&emsp;这一步骤会初始化博客。

启动博客

hexo s

&emsp;启动博客以后，通过 localhost:4000 来访问自己的博客。访问以后，您将会看到hexo自带的文章。这篇文章介绍了如何使用hexo。

撰写新文章

hexo n "title"

&emsp;这一步骤会让hexo新建一篇名为 title 的文章。您可以前往source/_posts/目录下查看刚刚新建的文章，格式为Markdown。
