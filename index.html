关于**Freenom**域名停止注册这件事

Posted on 2023-05-13

原文标题：尴尬了 ┃ 全球唯一提供免费一级域名的Freenom，竟然被Facebook告了
转载自https://zhuanlan.zhihu.com/p/627211282

但凡做过网站的人都应该知道Freenom的免费域名。Freenom是全球唯一一家的免费的一级域名注册商，这家公司身处荷兰，现已停止允许新域名注册。

一、被FACEBOOK起诉

原因竟然是Meta （更名后的facebook）指控该公司无视关于网络钓鱼网站的滥用投诉，同时将这些滥用域的流量货币化。

二、无奈的Freenom，对外只能解释为技术问题

当你希望Freenom再次注册免费域名的时候，通常跳出这几行提示。

原文如下：

IMPORTANT NOTICE: Because of technical issues the **Freenom** application for new registrations is temporarily out-of-order.
Please accept our apologies for the inconvenience. We are working on a solution and hope to resume operations shortly.
Thank you for your understanding.

中文意思如下：

重要通知： 由于技术问题，新注册的 **Freenom** 申请暂时出现故障。
对于给您带来的不便，请接受我们的歉意。我们正在研究解决方案，希望尽快恢复运营。
感谢您的理解。

三、曾经免费的五个国际代码顶级域名

Freenom 是五个所谓的“国家代码顶级域名 ”(ccTLD)，包括
1、cf 中非共和国的；
2、ga代表加蓬；
3、gq代表赤道几内亚；
4、ml表示马里；
5、tk代表托克劳。

Freenom 一直免收这些国家代码域中的域的注册费，

大概是为了鼓励用户为相关服务付费，

例如注册 .com 或 .net 域，Freenom 确实会收取费用。

四、FACEBOOK起诉缘由

2023 年 3 月 3 日，社交媒体巨头 Meta 在北加州法院起诉 Freenom，指控其存在域名抢注和商标侵权行为。该诉讼还寻求有关 20 个不同“John Does”的身份的信息——Meta 表示，Freenom 客户在针对Facebook、Instagram和WhatsApp用户的网络钓鱼攻击中特别活跃。

该诉讼指向2021 年由Interisle Consulting Group进行的一项关于域名滥用的研究 (PDF) ，该研究发现由 Freenom 运营的 ccTLD 占网络钓鱼者滥用最多的十大 TLD 中的五个。

“Freenom 为其提供服务的五个 ccTLD 是网络犯罪分子选择的 TLD，因为 Freenom 提供免费域名注册服务并保护其客户的身份，即使在出示域名被用于非法目的的证据之后，”投诉费用。“即使在收到客户的侵权或网络钓鱼通知后，Freenom 仍继续向这些客户许可新的侵权域名。”

Meta 进一步声称，“Freenom 一再未能采取适当的措施来调查和适当回应滥用报告”，并且它通过转售侵权域名和添加“停放页面”将访问者重定向到其他商业网站，从而从中获利。 、包含色情内容的网站以及用于网络钓鱼等恶意活动的网站。

Freenom 尚未回应置评请求。但在发布时尝试通过公司网站注册域会生成一条错误消息，内容如下：

“由于技术问题，Freenom 的新注册申请暂时出现故障。对于给您带来的不便，请接受我们的歉意。我们正在研究解决方案，希望尽快恢复运营。谢谢你的理解。”

图片：Interisle Consulting Group，《2021 年网络钓鱼形势》，2021 年 9 月。

尽管 Freenom 总部位于荷兰，但在诉讼中被列为被告的其他一些姊妹公司却在美国注册成立。

Meta 最初于 2022 年 12 月提起这起诉讼，但它要求法院封存此案，这将限制公众查阅纠纷中的法院文件。该请求被拒绝，Meta 上周修改并重新提起诉讼。

根据 Meta 的说法，这不仅仅是另一个域名注册商忽视滥用投诉的案例，因为这对业务不利。该诉讼称，Freenom 的所有者“是为促进域名抢注而创建的公司网络的一部分，所有这些都是为了 Freenom 的利益。”

“根据信息和信念，创建了一个或多个 ccTLD 服务提供商、ID Shield、Yoursafe、Freedom Registry、Fintag、Cervesia、VTL、Joost Zuurbier Management Services BV 和 Doe Defentants 以隐藏资产，确保非法活动，包括域名抢注和网络钓鱼未被发现，并进一步实现了 Freenom 的目标，”Meta 指控道。

目前尚不清楚为什么 Freenom 已停止允许域注册。2015 年 6 月，ICANN 暂停了 Freenom 创建新域名或启动域名入站转移的能力 90 天。根据 Meta 的说法，暂停的前提是 ICANN 确定 Freenom“从事贩运或使用与注册名称持有人拥有的第三方商标或服务标记相同或混淆性相似的域名的模式和做法没有权利或合法利益。”

ICANN 发言人表示，该组织不了解 Freenom 为何停止注册域名。但它表示，Freenom（d/b/a OpenTLD BV）也在2017 年和 2020 年因违反不同义务而收到 ICANN 的正式执法通知。

如何使用Hexo搭建个人博客

Posted on 2022-11-26

前言

关于我使用Hexo搭建了个人博客以后，萌生了写作的念头。虽然小张经常骂我，但是我很生气我并没有恨他，因为他对我的编程之路有很大的帮助。今天，我就来跟风撰写一篇如何搭建个人Blog的文章。

前提

你需要安装好Nodejs以及Git.

正文

安装cnpm（淘宝源）

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo-cli

cnpm install -g hexo-cli

可以使用 hexo -v 来验证下。

新建文件夹

在指定位置新建文件夹，名字为blog。

初始化博客

hexo init

这一步骤会初始化博客。

启动博客

hexo s

启动博客以后，通过 localhost:4000 来访问自己的博客。访问以后，您将会看到hexo自带的文章。这篇文章介绍了如何使用hexo。

撰写新文章

hexo n "title"

这一步骤会让hexo新建一篇名为 title 的文章。您可以前往source/_posts/目录下查看刚刚新建的文章，格式为Markdown。

部署到远端（Github）上公开使用

1.登录自己的 Github。

2.新建一个仓库：

注意：一定是你的昵称.github.io

继续在 blog 目录下

去 blog 文件下，直接看到一个_config.yml，修改_config.yml即可。注意：blog文件夹下的其它文件下也有_config.yml文件，别改错了文件。

文件的最底部修改成这样：

deploy: type: git repository: https://github.com/qinghan586/qinghan586.github.io.git branch: master

注意：deploy: type: repository: branch: 后面都有一个英文的空格。

部署到远端：hexo d

往远端推成功后，刷新 Github 仓库，发现多了很多东西，这些就是博客内容。

下载好看的主题：

好看的 Hexo 主题地址：https://hexo.io/themes/

自己来换个主题，克隆到 themes 主题下去：

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

git clone https://github.com/volantis-x/hexo-theme-volantis.git themes/volantis

现在把主题配置到博客上去：

修改_config.yml文件：

theme: yilia

theme: volantis

你用哪个主题就换哪个主题的名字，注意名称前面有个英文空格。

输入以下命令：

hexo clean

hexo g

hexo s

推到远端（github）上去：hexo d

以上整个过程已经完成了。后续学下 css，然后才会把自己的博客修改的很漂亮。先搭建起来，再考虑博客内容美观的问题。

注意

修改_config.yml文件后，需要把主题配置到博客上去，输入

hexo clean

hexo g

hexo s

推到远端（github）上去：hexo d

都不行的情况下，可以换成这样的命令再次输入下：

npm install hexo-deployer-git --save

hexo clean

hexo generate

hexo deploy

最后，感谢您的阅读。
