如何使用Hexo搭建个人博客

Posted on 2022-11-26

前言

关于我使用Hexo搭建了个人博客以后，萌生了写作的念头。虽然小张经常骂我，但是我很生气我并没有恨他，因为他对我的编程之路有很大的帮助。今天，我就来跟风撰写一篇如何搭建个人Blog的文章。

前提

你需要安装好Nodejs以及Git.

正文

安装cnpm（淘宝源）

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo-cli

cnpm install -g hexo-cli

可以使用 hexo -v 来验证下。

新建文件夹

在指定位置新建文件夹，名字为blog。

初始化博客

hexo init

这一步骤会初始化博客。

启动博客

hexo s

启动博客以后，通过 localhost:4000 来访问自己的博客。访问以后，您将会看到hexo自带的文章。这篇文章介绍了如何使用hexo。

撰写新文章

hexo n "title"

这一步骤会让hexo新建一篇名为 title 的文章。您可以前往source/_posts/目录下查看刚刚新建的文章，格式为Markdown。

部署到远端（Github）上公开使用

1.登录自己的 Github。

2.新建一个仓库：

注意：一定是你的昵称.github.io

继续在 blog 目录下

去 blog 文件下，直接看到一个_config.yml，修改_config.yml即可。注意：blog文件夹下的其它文件下也有_config.yml文件，别改错了文件。

文件的最底部修改成这样：

deploy: type: git repository: https://github.com/qinghan586/qinghan586.github.io.git branch: master

注意：deploy: type: repository: branch: 后面都有一个英文的空格。

部署到远端：hexo d

往远端推成功后，刷新 Github 仓库，发现多了很多东西，这些就是博客内容。

下载好看的主题：

好看的 Hexo 主题地址：https://hexo.io/themes/

自己来换个主题，克隆到 themes 主题下去：

git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

git clone https://github.com/volantis-x/hexo-theme-volantis.git themes/volantis

现在把主题配置到博客上去：

修改_config.yml文件：

theme: yilia

theme: volantis

你用哪个主题就换哪个主题的名字，注意名称前面有个英文空格。

输入以下命令：

hexo clean

hexo g

hexo s

推到远端（github）上去：hexo d

以上整个过程已经完成了。后续学下 css，然后才会把自己的博客修改的很漂亮。先搭建起来，再考虑博客内容美观的问题。

注意

修改_config.yml文件后，需要把主题配置到博客上去，输入

hexo clean

hexo g

hexo s

推到远端（github）上去：hexo d

都不行的情况下，可以换成这样的命令再次输入下：

npm install hexo-deployer-git --save

hexo clean

hexo generate

hexo deploy

最后，感谢您的阅读。
